To create a release, tag the commit:
   git commit -m 'some commit message'
   git tag -a v2.0.0.2 -m 'new shiny release' 2.0.0.2
   git push --tags

To delete a tag:
   git tag -d 2.0.0.2

To see tags: ('-n' shows comment, '-l' shows in order)
   git tag -n -l
